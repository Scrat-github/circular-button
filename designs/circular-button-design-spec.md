# 圆形按钮设计说明文档

## 设计概述

现代风格圆形按钮组件，采用径向渐变背景，支持三种交互状态。

---

## 组件结构

```
CircularButton
├── Button Container (圆形框架)
│   ├── Gradient Background (径向渐变填充)
│   └── Icon Container (图标容器)
│       └── SVG Star Icon (星形图标)
└── Shadow Layer (阴影层 - 根据状态变化)
```

---

## 交互状态详解

### 1. 默认态 (Default)
- 按钮保持基础尺寸 120px
- 显示基础阴影
- 光标：default

### 2. 悬停态 (Hover)
- 按钮放大至 132px (1.1x)
- 阴影加深增强悬浮感
- 颜色向紫色端偏移
- 光标：pointer

### 3. 点击态 (Active)
- 按钮缩小至 108px (0.9x)
- 阴影减弱模拟按下效果
- 颜色变为深紫色
- 光标：pointer

---

## 无障碍设计

- **键盘支持**: Tab 键可聚焦
- **焦点样式**: 2px 白色外环 + 2px 蓝紫色内环
- **ARIA 标签**: `aria-label="圆形操作按钮"`
- **触摸目标**: 最小 44x44px (移动端)

## 设计原则

1. **简洁性**: 单一圆形几何体，无多余装饰
2. **一致性**: 三种状态保持视觉连贯性
3. **反馈性**: 每个交互状态都有明确的视觉反馈
4. **可访问性**: 符合 WCAG 对比度要求
5. **响应式**: 三色断点适配移动端
